# 判断

## 关系运算符

| 运算符 | 意义 |
| :----: | :--: |
| `==`   | 等于 |
| `!=` | 不等于 |
| `>`  | 大于 |
| `>=` |大于等于|
| `<`  |小于|
| `<=` |小于等于|



```java
System.out.println(1<2);//true
```

## if-else

```java
if (x>y){
    max = x;
}
else{
    max = y;
}
```

## switch-case

    case value :
