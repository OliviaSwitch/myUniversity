## 现在分词
### 变位
即直陈式现在时第一人称复数 nous 的变位去掉 ons 再加上 ant，如：

| nous aim~~ons~~    | ->  | aimant    |
| ------------------ | --- | --------- |
| nous finiss~~ons~~ | ->  | finissant |
| nous lis~~ons~~    | ->  | lisant    |
| nous écriv~~ons~~  | ->  | écrivant  |

#### 特殊变位
| être   | ->  | étant   |
| ------ | --- | ------- |
| avoir  | ->  | ayant   |
| savoir | ->  | sachant |

### 用法
常见于书面用语

## 副动词
### 变位
即 en+现在分词变位
### 用法
书面语与口语都常见
